# AJF Automáticos — Parte 1

Estrutura inicial com:
- cabeçalho fixo com blur
- menu desktop e mobile
- primeira sessão hero
- botão neon com brilho e animação de escala
- arquivos separados para subir no GitHub

## Estrutura

```text
ajf-site-parte1/
├── index.html
├── css/
│   └── styles.css
├── js/
│   └── script.js
└── assets/
    ├── hero-desktop.webp
    ├── hero-mobile.webp
    ├── logo.webp
    └── whatsapp-icon.webp
```

## Observações

1. A fonte está puxando via Google Fonts com `Teko` peso 600, que é o equivalente visual mais próximo do `Teko Semibold`.
2. O link do WhatsApp está com número placeholder em dois pontos do HTML. Troque por seu número real antes de publicar.
3. As seções `#sobre`, `#servicos` e `#cases` ficaram como âncoras-base para você expandir nas próximas etapas.
